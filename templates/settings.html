카카오 앱 설정 가이드
  1. developers.kakao.com 접속 후 로그인 → 내 애플리케이션 → 애플리케이션 추가하기
  2. 앱 이름: 청약알리미 (마음대로), 회사명: 본인 이름 입력
  3. 앱 설정 → 플랫폼 → Web 추가:
    http://localhost:5000
  4. 제품 설정 → 카카오 로그인 → 활성화 ON
    Redirect URI 추가: http://localhost:5000/kakao/callback
  5. 카카오 로그인 → 동의항목 → 카카오톡 메시지 전송(talk_message) 설정 선택동의로 설정
  6. 앱 설정 → 앱 키 → REST API 키 복사 후 위에 입력
  7. 카카오 로그인 버튼 클릭 → 권한 허용 → 완료!
